# Dino Crisis 2 — Archipelago world (STUB)

A Python [Archipelago](https://archipelago.gg) world that lets AP **generate** a Dino Crisis 2 seed.
It is a deliberate **stub**: DC2 has a decoded door-graph + item spots (`data/dc2`) but no key-item
ids or door→key gating yet (KaQ OPEN #2/#5), so there is no logically-gated graph to shuffle. The
world builds whatever the contract carries (empty today) plus one free **Victory** event so AP still
produces a completable seed. See `docs/decisions/cross/ARCHIPELAGO-DC2-STUB-PLAN.md`.

Enabling real DC2 rando later is **populating data, not rewriting code**: a DC2 builder fills
`data/dc2_logic.json` (regions/locations/edges/items) and `world.py` lights up unchanged.

## Layout
| File | Role |
|------|------|
| `world.py` | `DinoCrisis2World` — contract-driven regions/locations/items + a Victory event |
| `data.py` | loads `data/dc2_logic.json`, builds name→id maps (empty in the stub) |
| `options.py` | `DinoCrisis2Options` (empty stub — required by AP) |
| `data/dc2_logic.json` | the contract, generated by `scripts/gen_ap_logic.py dc2` (do not hand-edit) |
| `archipelago.json` | AP world manifest |
| `test/` | AP-side integration tests (`WorldTestBase`) — run inside an AP checkout |
| `selfcheck.py` | AP-independent structural check — runs anywhere, no AP install |

## Generate a seed with an Archipelago checkout
```bash
git clone https://github.com/ArchipelagoMW/Archipelago && cd Archipelago
python ModuleUpdate.py
cp -r /path/to/dinorand/apworld/dino_crisis_2 worlds/
cat > Players/DC2.yaml <<'YAML'
name: DC2Player
game: Dino Crisis 2
Dino Crisis 2: {}
YAML
python Generate.py --seed 1
```
Success = `output/AP_*.zip` written with no `FillError` (the spoiler shows the free Victory event).

## Tests
AP-side (from the Archipelago root, after copying the world in):
```bash
python -m pytest worlds/dino_crisis_2/test
```
AP-independent (from the DinoRand repo, no AP install):
```bash
python3 scripts/gen_ap_logic.py --check      # every game's contract in sync
python3 apworld/dino_crisis_2/selfcheck.py   # world.py structural wiring via a minimal AP stub
```

## Regenerating the logic contract
```bash
python3 scripts/gen_ap_logic.py dc2 --apply   # regenerate + stage (or omit `dc2` for all games)
```
