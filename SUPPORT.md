# Support

DinoRand supports only the latest published release and the GOG DRM-free editions of Dino Crisis 1
and 2. Steam/Enigma-protected builds are unsupported. Check the [User Guide](USER-GUIDE.md), restore
the original files, and reproduce with the latest release before filing a bug.

Use the DinoRand bug form for reproducible product problems and the feature form for proposals. We
target an initial acknowledgment within seven days, but do not promise a fix deadline.

Do not upload game files, executables, copyrighted-byte dumps, credentials, or private data. Report
security vulnerabilities privately as described in [SECURITY.md](SECURITY.md), not in a public issue.
